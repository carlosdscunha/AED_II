[S7AP2] Manipulação de arquivos de texto

    Crie, em C, um programa que permite registrar (usando estruturas de dados heterogêneas) uma conjunto de vídeos do YouTube. Para cada vídeo, devem ser armazenadas as seguintes informações: título, nome do canal, link e duração (em segundos). Ao ser executado, o programa deve persistir os dados na memória secundária, especificamente por um arquivo de texto. Neste arquivo, os dados de cada entidade devem aparecer separados por vírgula. 

    Depois, em um segundo programa, implemente um mecanismo capaz de recuperar os vídeos listados no arquivo gerado pelo programa anterior. Este programa deve, ao iniciar, imprimir todos os registros que encontrar no banco.

    Após a impressão de todos os registros, o programa deve exibir as seguintes opções para os usuários:

        a) listar apenas vídeos com duração superior a 120 segundos;

        b) listar apenas os vídeos com duração de até 120 segundos;

    Naturalmente, após selecionar cada opção, o programa deve exibir os registros equivalentes.
