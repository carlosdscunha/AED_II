[S7AP1] Manipulação de arquivos binários

    Implemente um programa em C que é capaz de armazenar, pelo menos, cinco registros de uma determinada entidade (definida por você). O programa deve gravar os dados das cinco ocorrências dessa entidade em um arquivo binário com extensão .txt. Depois, implemente um segundo programa em C, capaz que ler o arquivo criado pelo programa anterior. Assim que executado, o programa dever ler o arquivo e imprimir os dados da entidade armazenada.
